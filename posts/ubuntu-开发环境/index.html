    Ubuntu 开发环境

    May 28, 2022

    目标#

    能够完成日常开发所需

    软件相关#

    Ubuntu 系统下载

    Ventoy 主力启动盘工具，里面可以同时放多个 ISO 也可以嵌套启动工具比如里面还可以放老毛桃和 WEPE

    Wepe 备用启动盘工具

    Oh my zsh， Terminal 强化, 二级命令提示 UI 优化

    Starship 和 Oh my zsh 功能有重合的部分，UI 上会更友好，可以二选一，也可以都用

    fonts-font-awesome 解决图标类型的字体，例如 Git 分支标，Java， Maven 等图标，主要由 Oh my zsh 和 Starship 引入

    VS Code 开发工具和文本编辑器，适合 Go 语言开发

    IDEA Java 开发工具

    GoLand Go 语言开发工具

    datagrip

    ja-netfilter Jetbrin 激活工具

    Sublime Text4 文本编辑，速度非常快，插件也很丰富

    Sublime Merge git GUI 工具

    Chrome 主力浏览器

    Microsoft Edge 备用浏览器

    Brave brwoser 备用浏览器

    To Desk 远程桌面链接工具

    Netspeed 状态栏网速显示插件

    Koodo Reader 电子书阅读，选择翻译实用

    VLC视频播放软件

    NetHogs 查看进程使用的流量

    aria2 命令行下载工具

    htop 增强 top 命令提供更多详细信息

    httpie UI 增强型 cURL

    hugo Blog 构建工具，本网页就是用 hugo 构建的

    docker 虚拟化技术，快速和运行一些服务 Nginx, Redis 等

    docker portainer Docker web UI dashboard

    obs-studio 录屏，虚拟化摄像头等

    slack 聊天软件

    Telegram 聊天软件

    Discord 聊天软件

    Spotify 音乐软件

    Postman 接口连调工具

    RedisInsight redis 官方出品的 GUI 工具

    Clash 代理工具

    Flameshot 截图工具

    内置软件#

    iftop 查看浏览信息，以域名进出为主和 Nethogs 相互补充

    iotop 查看磁盘读写 情况

    remmina ssh,ftp 链接工具

    vim 文本编辑

    问题解决#

    • Jetbrain IDE 打中文会造成无法打字的情况
     在 xxx.vmproperteies 中添加 `-Drecreate.x11.input.method=true`

https://youtrack.jetbrains.com/issue/JBR-2444
    • OBS 虚拟摄像头只能打开一次问题解决
     https://github.com/obsproject/obs-studio/issues/4808#issuecomment-1108880336
    • 使用 zsh 后环境变量需要添加在 ~/.zshrc 中，例如：
    export HTTP_PROXY=http://127.0.0.1:7890
export HTTPS_PROXY=$HTTP_PROXY
export PATH=$PATH:/usr/local/go/bin
export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64/


eval "$(starship init zsh)"
___MY_VMOPTIONS_SHELL_FILE="${HOME}/.jetbrains.vmoptions.sh"; if [ -f "${___MY_VMOPTIONS_SHELL_FILE}" ]; then . "${___MY_VMOPTIONS_SHELL_FILE}"; fi
